# TransitOps 🚚

**TransitOps** is a modern, unified fleet management platform designed to help logistics companies efficiently run vehicles, drivers, trips, maintenance, and analytics from a single dashboard.

## 🚀 Live Demo

**[View the Live Application](https://transitops-wine.vercel.app/)**

- **Password:** `password123`

## 🌟 Key Features

- **Dashboard & Analytics:** Get a high-level overview of active trips, vehicle utilization, and maintenance schedules.
- **Vehicle Management:** Track vehicle statuses, maintenance records, and fuel logs.
- **Driver Management:** Manage driver profiles, licenses, and assignments.
- **Trip Tracking:** Monitor ongoing trips, origins, destinations, and costs.
- **Expense Tracking:** Keep an eye on fuel costs, maintenance bills, and overall fleet expenditure.

## 🛠️ Technology Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS, shadcn/ui, Lucide Icons
- **Backend:** Node.js, Express, TypeScript, MongoDB (Mongoose)
- **Deployment:** Vercel (Frontend SPA), Render (Backend API), MongoDB Atlas (Database)

## 💻 Local Development

### 1. Backend Setup
```bash
cd backend
npm install

# Create a .env file and add your MONGO_URI
# Run the seed script to create the initial admin user
npm run seed

# Start the development server
npm run dev
```

### 2. Frontend Setup
```bash
npm install

# Start the Vite development server
npm run dev
```
